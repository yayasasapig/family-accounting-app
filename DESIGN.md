**版本：** v1.0

## 1. 設計理念

### 1.1 核心願景

> 「溫暖如家，易如掌中」

我哋唔係做銀行 App，我哋係做紧一個屬於香港兩公婆嘅家庭記帳工具。要溫暖、和諧、清晰，唔要冷冰冰嘅數字報告風格。

### 1.2 設計原則

| 原則 | 描述 |
|------|------|
| **溫暖感** | 用柔和色彩同圓潤造型，唔要有尖銳嘅綫條 |
| **清晰優先** | 一眼睇得明，唔使解釋 |
| **本地化** | 香港元素（港幣、繁體中文、粵語習慣） |
| **Mobile-first** | 兩公婆都係用手機为主 |
| **信任感** | 穏定、可靠、家人之間共享無壓力 |

## 6. 動效規範

### 6.1 過渡時間

| 動作 | 時長 |
|------|------|
| 按鈕 Hover | 150ms |
| 頁面切換 | 250ms |
| 卡片展開 | 200ms |
| Modal 彈出 | 300ms |
| 載入中 | 800ms（循環） |

### 6.2 常用動效

- 按鈕 Hover：背景色加深 10%，輕微上浮（translateY -2px）
- 卡片 Hover：陰影增強，輕微放大（scale 1.01）
- 頁面切換：由右滑入（translateX 100% → 0）
- 刪除：向左滑出 + 淡出
- 新增收支：卡片由下向上生長（scaleY 0 → 1）

## 10. Mobile-First 響應式設計

### 10.1 斷點

| 名稱 | 寬度 | 佈局 |
|------|------|------|
| Mobile | < 640px | 單欄，全寬卡片 |
| Tablet | 640px - 1024px | 雙欄，卡片網格 |
| Desktop | > 1024px | 多欄，側邊導航 |

### 10.2 Mobile 優先原則

1. 優先保證 Mobile 體驗
2. 單欄佈局適合手指操作
3. 底部 Tab Bar 方便單手操作
4. 按鈕最小高度 48px
5. 預留拇指區域（右下角 FAB）
